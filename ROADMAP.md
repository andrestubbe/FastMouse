# FastMouse ROADMAP

Hier ist der Entwicklungsplan für FastMouse:

| Version | Ziel / Features | Status |
| :--- | :--- | :--- |
| **v0.1.0** | Raw Input WM_INPUT (Mausbewegung, Clicks, Wheel) | **Fertig** |
| **v0.2.0** | Absolute Pixel-Koordinaten via Win32 `GetCursorPos` | **In Arbeit** |
| **v0.3.0** | Scroll Wheel High-Resolution (Precision-Scrolling Unterstützung) | Geplant |

| **v0.5.0** | Linux evdev Treiber-Support (Platform-Parität) | Geplant |
| **v1.0.0** | Zero-GC API Stabilisierung & JitPack Release | Geplant |
